# Open Agent Harness

A growth-first harness framework for human and AI collaboration quality.

Works with Goose.

## Why this exists
Most agent tooling optimizes for raw throughput. Open Agent Harness optimizes for collaboration maturity, developmental growth, reliable quality, and cost-aware execution.

## Core idea
The framework provides a frame, not a personal workflow.
Teams define their own profiles, conventions, and toolchains.

## Focus
- collaboration maturity progression
- reliable validation and convention alignment
- lower avoidable CI failures
- higher quality with low-cost operating patterns
