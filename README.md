# Open Agent Harness

Open Agent Harness is a growth-first framework for human and AI collaboration quality.

Works with Goose.

## Packages
- create-agent-harness

## Install
npm:
- npm create @dakotafabro/agent-harness my-harness

bun:
- bun create @dakotafabro/agent-harness my-harness

## Focus
- collaboration maturity
- quality consistency
- learning and reflection loops
- cost-aware execution

## Design boundary
The framework provides a reusable frame. It does not ship a personal workflow. Teams define their own profile, conventions, and tools.
