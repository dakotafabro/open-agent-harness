# Core Concepts

## Harness
A harness is the control layer around AI execution. It defines how work is prepared, constrained, verified, and reflected on.

## Contract
A contract is a structured agreement for behavior. Examples include policy contracts, workflow contracts, and output contracts.

## Profile
A profile is a team-specific implementation of the framework. It contains tools, conventions, and quality gates for a local environment.

## Validation Gate
A gate is a checkpoint that must pass before advancing phases. Gates reduce expensive rework.

## Reflection Artifact
A reflection artifact captures what improved quality, what failed early, and what should become reusable practice.

## Collaboration Maturity
Collaboration maturity measures quality of interaction between human and AI over time, not only output quantity per hour.
