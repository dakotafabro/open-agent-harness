# Launch Draft - Open Agent Harness

Open Agent Harness is now live as an installable scaffold and CLI for growth-first human and AI collaboration.

Works with Goose.

## What launched
- @dakotafabrodev/create-agent-harness for project scaffolding
- @dakotafabrodev/agent-harness for validate and run commands
- robust documentation covering quickstart, concepts, architecture, maturity model, adoption, and troubleshooting

## Why this matters
Most agent tooling optimizes for speed first. Open Agent Harness optimizes for collaboration maturity, quality consistency, and developmental growth. Teams can move faster as an outcome of better structure and clearer gates, while keeping costs disciplined.

## What this unlocks
- a reusable frame that does not encode one personal workflow
- policy and workflow contracts teams can adapt to their own stack
- earlier failure detection through validation, reducing avoidable CI rework
- reflection loops that convert experience into reusable practice

## Proof strategy for quality and evidence bonuses
- run one pilot repo for 2 to 4 weeks
- capture baseline and outcome metrics using docs/pilot-metrics-template.md
- publish one evidence snapshot with links to artifacts
- include one concrete prevented rework example

## Try it
- npx @dakotafabrodev/create-agent-harness my-harness
- npx @dakotafabrodev/agent-harness validate
- npx @dakotafabrodev/agent-harness run build
