# Release Announcement - v0.1.1

Open Agent Harness v0.1.1 is live.

Works with Goose.

## Highlights
- Added guided scaffold setup for faster onboarding
- Added SETUP.md to every generated project
- Added folder-level guidance docs for all customizable harness areas
- Kept defaults generic so teams can define their own profile and conventions

## Install
- npx @dakotafabrodev/create-agent-harness my-harness

Optional create aliases:
- npm create @dakotafabrodev/agent-harness my-harness
- bun create @dakotafabrodev/agent-harness my-harness

## Run
- npx @dakotafabrodev/agent-harness validate
- npx @dakotafabrodev/agent-harness run build

## Why this release
This update improves first-run clarity and helps teams customize safely without personal workflow lock-in.
